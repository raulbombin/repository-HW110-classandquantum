# Data and plotting scripts for

**Quantum isotope effects in the scattering of hydrogen isotopes from W(110): the role of phonon-assisted absorption**

Preprint: https://arxiv.org/pdf/2603.05426

## Overview

This repository contains the data and Python scripts used to generate the figures presented in the accompanying paper. The repository includes both quantum (MCTDH) and classical dynamics results for the scattering of H, D, and T atoms from the W(110) surface.

The scripts reproduce the figures directly from the data files contained in this repository.

## Description of the datasets

### `data-potfitcuts/`

Potential-energy cuts and minimum-energy-path data used to visualize the fitted potential-energy surface.

The files `cutXY-Z*.xyz` contain XY cuts of the potential at the Z value indicated in the filename. The files `cutZ-*` contain cuts of the potential along the Z direction at the surface position indicated in the filename. Distances are given in bohr and energies in eV.

The file `MEP.dat` contains five columns. The first three correspond to the X, Y, and Z coordinates, the fourth contains the corresponding potential energy, and the fifth contains the cumulative distance along the minimum-energy path.

These data are used by the following script to generate Fig. 1 of the paper:

```
plot-potential.py
```

---

### `data-PabssvsEin-H/`

Quantum absorption probabilities as a function of incident energy for hydrogen. The data correspond to individual MCTDH calculations performed at different incident energies.

These files are generated by the `flux84` analysis program included in the Heidelberg MCTDH package. Each file contains four columns:

1. Incident energy.
2. Energy-resolved flux through the absorption CAP.
3. Incident energy distribution.
4. Energy-resolved absorption probability.

This directory also contains the classical reference curve in the file `Pabs_vs_Ein_classical.dat`, which contains three columns corresponding to the incident energy, the absorption probability, and the reflection probability.

These data are used by the following script to generate Fig. 2 of the paper:

```
plot-PabsvsEin-H.py
```

---

### `data-PabssvsEin-isotopes/`

The files `Pabs_vs_Ein_*_quantum.dat` contain the absorption probability curves for H, D, and T calculated using MCTDH.

This directory also contains the classical reference curve in the file `Pabs_vs_Ein_classical.dat`, which contains three columns corresponding to the incident energy, the absorption probability, and the reflection probability.

Files:

* `Pabs_vs_Ein_classical.dat`
* `Pabs_vs_Ein_H_quantum.dat`
* `Pabs_vs_Ein_D_quantum.dat`
* `Pabs_vs_Ein_T_quantum.dat`

These data are used by the following script to generate Fig. 3 of the paper:

```
plot-PabsvsEin-isotopes.py
```

---

### `data-Prefvstime/`

Time-dependent reflection probabilities for H, D, and T obtained from classical and quantum calculations at the following incident energies:

* 0.050 eV
* 0.800 eV

The files `Pref-classical*` contain the time evolution of the classical absorption and reflection probabilities.

The files `Pref-quantum*` are generated by the `flux84` analysis program included in the Heidelberg MCTDH package. Each file contains four columns:

1. Incident energy.
2. Energy-resolved flux through the reflection CAP.
3. Incident energy distribution.
4. Energy-resolved reflection probability.

These data are used by the following script to generate Fig. 4 of the paper:

```
plot-Prefvstime-isotopes.py
```

---

### `data-diffraction/`

Angular diffraction probabilities for classical and quantum calculations for H, D, and T at the following incident energies:

* 0.050 eV
* 0.100 eV
* 0.300 eV

Each `diffraction-classical*` file contains four columns:

1. Diffraction order.
2. Diffraction energy.
3. Probability obtained using Gaussian binning.
4. Probability obtained using simple binning.

The energy is given in hartree.

Each `diffraction-quantum*` file contains two columns corresponding to the diffraction energy and the probability associated with that diffraction channel.

These data are used by the following script to generate Fig. 5 of the paper:

```
plot-diffraction.py
```

---

### `data-WFs/`

The WF* files contain the  XY cuts of the of the ground state and excited states at Z=1.1 angstrons from the surface.

These data are used by the following script to generate Fig. 6 of the paper:

```
plot-WFs.py
```

---

## Reproducing the figures

The figures can be regenerated by executing

```bash
python plot-diffraction.py
python plot-PabsvsEin-H.py
python plot-PabsvsEin-isotopes.py
python plot-Prefvstime-isotopes.py
python plot-potential.py
python plot-Wfs.py
```

The scripts require

* Python 3
* NumPy
* Matplotlib
